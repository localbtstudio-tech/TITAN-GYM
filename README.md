## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/localbtstudio-tech/titan-gym.git
```

Navigate to the project

```bash
cd titan-gym
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
